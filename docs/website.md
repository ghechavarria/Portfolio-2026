# Portfolio Homepage

## Files
- `index.html`: single-page portfolio structure updated with resume and LinkedIn content.
- `styles.css`: visual styling for the dark neon layout, fixed pill nav, theme toggle, curved hero, cards, footer links, and motion effects.

## Notes
- Uses local SVG assets from `assets/` for decorative section icons.
- Designed as a static starter that now includes the language, systems, certification, date details, hover-reveal role summaries, a stacked experience strip under Projects, and an education block with the diploma icon from Grace Hechavarria's resume and LinkedIn profile.
